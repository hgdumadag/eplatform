# Main Content

## What is Addition?

Addition means **putting things together** or **combining groups**. When we add, we're counting how many things we have in total.

The **plus symbol** (+) tells us to add. When you see $2 + 3$, you read it as "two plus three."

## Understanding the Parts

Every addition problem has three important parts:

1. **First Addend** - the first number
2. **Second Addend** - the second number
3. **Sum** - the answer (the total)

For example, in the problem $4 + 5 = 9$:
- $4$ is the first addend
- $5$ is the second addend
- $9$ is the sum (our answer!)

## Counting On

One easy way to add is to **count on** from the first number. Let's try adding $6 + 3$:

1. Start with the bigger number: **6**
2. Count on 3 more: **7, 8, 9**
3. The answer is **9**!

## Using Your Fingers

Your fingers are great tools for addition! If you need to add $3 + 4$:
- Hold up 3 fingers on one hand
- Hold up 4 fingers on the other hand
- Count all your fingers: 1, 2, 3, 4, 5, 6, 7
- The answer is **7**!

# Formulas

## Basic Addition Formula

$$a + b = c$$

Where:
- $a$ is the first addend
- $b$ is the second addend
- $c$ is the sum (total)

## Order Doesn't Matter!

This is called the **Commutative Property** of addition. It means you can add numbers in any order:

$$2 + 5 = 5 + 2$$

Both equal $7$!

## Adding Zero

Adding zero to any number gives you the same number:

$$7 + 0 = 7$$

This is called the **Identity Property** of addition.

# Examples

## Example 1: Simple Addition

**Problem:** What is $2 + 3$?

**Solution:**
- Start with 2
- Count on 3 more: 3, 4, 5
- **Answer: $2 + 3 = 5$**

## Example 2: Larger Numbers

**Problem:** Add $4 + 6$

**Solution:**
- Start with 4
- Count on 6 more: 5, 6, 7, 8, 9, 10
- **Answer: $4 + 6 = 10$**

## Example 3: Word Problem

**Problem:** Sarah has 5 crayons. Her teacher gives her 2 more crayons. How many crayons does Sarah have now?

**Solution:**
- Sarah starts with 5 crayons
- She gets 2 more crayons
- We need to add: $5 + 2$
- Count on from 5: 6, 7
- **Answer: Sarah has $7$ crayons!**

## Example 4: Using the Commutative Property

**Problem:** Show that $3 + 8$ equals $8 + 3$

**Solution:**
- $3 + 8 = 11$ (count: 4, 5, 6, 7, 8, 9, 10, 11)
- $8 + 3 = 11$ (count: 9, 10, 11)
- **Both equal $11$! Order doesn't matter!**

## Example 5: Adding Zero

**Problem:** What is $9 + 0$?

**Solution:**
- When you add zero, the number stays the same
- **Answer: $9 + 0 = 9$**
